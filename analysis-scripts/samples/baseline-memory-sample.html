01 Executive Summary

During forensic analysis of the baseline memory image captured from BASE-RD-01 (the Remote Desktop Server at 172.16.6.11) on 2018-09-06 at 18:57:17 UTC, one non-standard process was identified with strong indicators of malicious activity.

A single-character executable named p.exe, stored in a Windows Temp subdirectory, was found executing via a WMI-spawned PowerShell chain — a technique consistent with state-level APT tradecraft. Memory analysis of its process space revealed sequential IP addresses spanning the entire Business Line subnet (172.16.7.0/24), indicating active internal reconnaissance. The process binary was scrubbed from memory (process-hollowing indicator).

A second non-standard process, subject_srv.exe, was confirmed as a legitimate F-Response forensic collection agent deployed by the incident response team.
